# modelo-rs-vue
Aplicação Vue para consulta de modelos de RS e Tarefas.

## Dependênicas
Rodar a api [modelo-rs-api](https://github.com/igorgcosta/modelo-rs-api)

## Instalação
Após clone, executar:
```sh
cd modelo-rs-vue
```
Para instalar dependências:
```sh
npm install
```
Para rodar em desenvolvimento:
```sh
npm run dev
```
Para rodar em produção:
```sh
npm start
```
